Data List Builder for InDesign

Easily build a list of items to import into InDesign using this step-by-step tool. No spreadsheet knowledge required.

Step 1: What are you creating a list of?

Type a name that describes your items, e.g., "Products", "Team Members", or "Event Sessions".

Step 2: Add the details you want for each item

Each field is a column in your list — like "Name", "Price", or "Email".

Step 3: Fill in your data

Add one item per row. You can add more rows if needed.
